What is the difference between local storage and session storage

Local Storage : Their main difference is that local storage data does not expire, whereas session storage data gets cleared when the user closes its browser window.
Sessions Storage : Page sessions are valid for only one tab at a time. So, if a user has three tabs open, each one will have its own page session.Since local storage is not session-based, all the data must be deleted via javascript or manually.

What is the difference between global scope and block scope

Block Scope : A block scoped variable means that the variable defined within a block will not be accessible from outside the block.
Global Scope : Variables declared Globally (outside any function) have Global Scope. Global variables can be accessed from anywhere in a JavaScript program.
